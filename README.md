# 🎓 UOLi-HUB

> Your Ultimate Student Companion for Academic Success 🚀

UOLi-HUB is a comprehensive student portal designed to simplify academic life by providing essential tools, resources, and information in one place.

## 🌟 Overview

UOLi-HUB helps students manage their academic activities efficiently. Whether you need to create assignment front pages, calculate your CGPA, explore department information, or access course syllabi, UOLi-HUB has you covered.

---

## ✨ Features

### 📄 Assignment Front Page Generator

Create professional assignment front pages instantly with a simple and user-friendly interface.

### 📊 CGPA Calculator

Calculate your Semester GPA and Overall CGPA accurately and quickly.

### 🏛️ Department Information

Access detailed information about different university departments.

### 📚 Course Syllabus

View and download semester-wise syllabi for various academic programs.

### 💻 Programming Languages Hub

Learn about different programming languages, their features, uses, and learning resources.

### 🎯 Academic Resources

Access useful study materials, guides, and educational content.

### 📱 Responsive Design

Fully optimized for desktop, tablet, and mobile devices.

### ⚡ Fast and Easy to Use

Simple interface designed specifically for students.
